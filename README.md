# Partnerguiden: Klimakteriet

En webbsajt som hjälper partners att förstå klimakteriet – vad som händer, varför, och hur man kan vara ett bra stöd.

🔗 **[partnerguiden.se](http://partnerguiden.se)**

## Om projektet

Partnerguiden är ett ideellt projekt skapat för att ge partners konkret kunskap om klimakteriet. Genom korta artiklar, quizfrågor och dagliga mejlpåminnelser guidas läsaren genom ämnet i sin egen takt.

## Funktioner

- 📚 **Artikelserie** – 13 artiklar som täcker allt från hormoner till relationer
- ❓ **Quizfrågor** – kunskapstest efter varje artikel
- 📊 **Framstegshantering** – skapa konto och följ din progress
- 📧 **Dagliga mejlutskick** – automatiska påminnelser med nästa artikel
- 🔧 **Admin-panel** – hantera artiklar och FAQ:er
- 🔍 **SEO-optimerad** – sitemap, meta-taggar och strukturerad data

## Teknikstack

- **Frontend:** React, TypeScript, Tailwind CSS, Vite
- **Backend:** Lovable Cloud (databas, autentisering, edge functions, mejlutskick)
- **Övrigt:** shadcn/ui, React Router, TanStack Query
